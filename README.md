# Lavneesh Rajput — Portfolio

Personal portfolio website built with React, Vite, Three.js, and Framer Motion.

## Live

**[https://8b3eb666-2f58-4ba2-9869-e4e05cd13cec-00-7pvp6un0huij.pike.replit.dev](https://8b3eb666-2f58-4ba2-9869-e4e05cd13cec-00-7pvp6un0huij.pike.replit.dev)**

## Tech Stack

- **React + Vite** — fast, modern frontend tooling
- **Three.js** — animated WebGL background with custom GLSL shaders (simplex noise, mouse parallax, chromatic glow); CSS-animated fallback for environments without GPU
- **Framer Motion** — entrance animations and hover interactions
- **Tailwind CSS** — utility-first styling with a dark navy + electric violet theme
- **Space Grotesk** — typography
- **pnpm workspaces** — monorepo structure

## Sections

- **Hero** — name, role, CTA buttons
- **About** — bio, skills grouped by category (Back-end, Front-end, Tools & Cloud, CS Fundamentals)
- **Projects** — Twitter Clone, E-Commerce Platform, SaaS Risk Monitor (work), Healthcare Kanban (work)
- **Contact** — email, GitHub, LinkedIn, LeetCode links

## Run locally

```bash
pnpm install
pnpm --filter @workspace/portfolio run dev
```
